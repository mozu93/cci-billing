# 新規請求書の発行フロー

## 前提：初回のみ必要な設定

**設定タブ** で以下を登録しておきます：

1. **発行元情報** → 商工会議所名・住所・電話・T番号（登録番号）・銀行口座
2. **カテゴリ** → 例：「青年部」「女性部」「検定」
3. **請求項目テンプレート** → 例：「青年部会費 ¥10,000/式」（税率・書類種別も設定）
4. **会員マスタ** → ExcelインポートまたはTSV貼り付けで会員を一括登録

---

## ケース①：会員名簿がある定例請求（青年部会費など）

**典型例：** 毎年度、青年部の全会員に会費の請求書を発行する場合

### 手順

1. **事業管理タブ** → 「＋ 新規事業」ボタン
2. 事業名「2026年度 青年部会費」、カテゴリ「青年部」、種別「リスト型」を入力
3. テンプレートで「青年部会費 ¥10,000」を選択 → 保存
4. 作成した事業を選択すると下半分に会員パネルが表示される
5. 「Excelインポート」または「貼り付けインポート」で会員を割当
6. 「受付開始（active）」ボタンで状態をactiveに変更
7. **発行タブ** → 「事業から発行」
8. 事業プルダウンで「2026年度 青年部会費」を選択
9. 会員一覧が表示される（未発行のみ表示）
10. 発行したい会員を選択 → 「準備（採番）」ボタン
    - ここで発行番号が採番される（例：INV-202605-0001）
11. 「発行する」ボタン → 配付方法を選択（窓口/郵送/メール）
    - PDFが自動生成されてビューアで表示される

---

## ケース②：複数の未払い請求をまとめて1枚で発行

**典型例：** 会費を払っていない人に、登録料と合算して1枚の請求書を作る場合

### 手順

1. **発行タブ** → 「人を検索して発行」
2. 事業所名やフリガナで会員を検索 → 会員を選択
3. その人の全事業の未発行一覧が表示される
4. まとめて請求したい項目にチェック（複数選択可）
5. 配付方法を選択 → 「選択した項目を発行する」
   - 合算した1枚のPDFが生成される

---

## ケース③：その場で宛先・枚数を入力して発行（窓口型）

**典型例：** そろばん検定の受験料を窓口でその場で受け取って領収書を発行する場合

### 手順

1. **事業管理タブ** → 「＋ 新規事業」→ 種別「窓口型」
2. テンプレートで「珠算検定受験料 ¥3,000/人」を選択 → 保存 → 受付開始
3. **発行タブ** → 「窓口型（即時発行）」
4. 事業「珠算検定」を選択
5. 事業所名「○○そろばん教室」を入力
6. 数量「3」（3人分）を入力 → 金額：¥9,000 が自動表示
7. 「発行する」→ PDFが生成されてビューアで表示される

---

## 発行後の流れ

### 入金消込

1. **発行タブ** → 「支払管理」
2. 事業を選択 → 対象の発行済み書類を選択
3. 「支払済みに更新」→ 入金日・入金額・方法を入力 → 記録

### 状況確認

- **ダッシュボード** → 事業ごとの「全件/発行済/支払済/未発行」が一覧表示
- **レポートタブ** → 「未払い一覧」で全事業横断の未回収を確認

---

## フロー早見表

| 状況 | 使うタブ・フロー |
|------|----------------|
| 名簿がある定例請求 | 事業管理で事業作成→会員割当 → 発行「事業から発行」 |
| 複数事業を合算発行 | 発行「人を検索して発行」 |
| 窓口でその場発行 | 発行「窓口型（即時発行）」 |
| 入金確認・消込 | 発行「支払管理」 |
| 未回収一覧 | レポート「未払い一覧」 |
